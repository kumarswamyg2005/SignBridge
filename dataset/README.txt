Dataset Instructions for SignBridge

1. WLASL Dataset (World Level American Sign Language):
URL: https://github.com/dxli94/WLASL
Instructions: Download WLASL_v0.3.zip, extract videos into /static/gestures/asl/ folder, rename each video file to the word it represents (e.g., hello.mp4, thanks.mp4)

2. MS-ASL Dataset (Microsoft):
URL: https://www.microsoft.com/en-us/research/project/ms-asl
Instructions: Request access and download. Organize into /static/gestures/asl/ folder same way.

3. For ISL (Indian Sign Language):
URL: https://www.kaggle.com/datasets/ahmedkhanak1995/sign-language-gesture-images-dataset
Instructions: Download and place in /static/gestures/isl/ folder

4. For Fingerspelling (A-Z letters):
URL: https://www.kaggle.com/datasets/grassknoted/asl-alphabet
Instructions: Download video clips for each letter, rename to a.mp4, b.mp4 ... z.mp4 and place in /static/gestures/asl/ folder
